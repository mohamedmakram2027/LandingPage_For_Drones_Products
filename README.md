# Drone Landing Page
This project is a modern landing page designed to showcase the latest drone products. 
The page focuses on presenting drones in an attractive and clean layout while ensuring a responsive design for different screen sizes.

## Features
- Modern and clean user interface
- Responsive design for different devices
- Product showcase section for drones
- Smooth layout using HTML and CSS

## Technologies Used
- HTML5
- CSS3

## Project Purpose
The purpose of this project is to practice front-end development skills by building a landing page for a technology product. 
It demonstrates the use of HTML for structure and CSS for styling and layout.

## How to Run the Project
1. Clone or download the repository.
2. Open the `index.html` file in any web browser.
